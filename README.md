🛒 BlinkIT Sales Dashboard

1. Project Title / Headline
📊 BlinkIT Sales Performance Dashboard – Retail Sales & Outlet Analytics
An interactive Power BI dashboard designed to analyze BlinkIT grocery sales performance, outlet efficiency, customer ratings, product categories, and inventory distribution across different outlet types and locations.         
                         
2. Short Description / Purpose
The BlinkIT Sales Dashboard provides a comprehensive overview of sales, customer ratings, inventory, and outlet performance. It helps business stakeholders monitor key metrics, identify top-performing product categories, and make data-driven decisions to improve sales and operational efficiency.                                                                        
3. Tech Stack
The dashboard was built using the following tools and technologies:
📊 Power BI Desktop – Used for creating interactive reports and visualizations.
🔄 Power Query – Data cleaning, transformation, and preparation.
🧠 DAX (Data Analysis Expressions) – Created KPIs, calculated measures, and dynamic insights.
🗂️ Data Modeling – Established relationships between sales, outlet, and product data.
📁 File Format – .pbix for development and .png for dashboard previews.                                                      
4. Data Source
Source: BlinkIT Grocery Sales Dataset
The dataset contains information related to:
Product sales and revenue
Outlet types and sizes
Outlet establishment year
Product categories
Customer ratings
Item visibility
Outlet locations
Inventory and item counts                                                                                                            
5. Features / Highlights
📌 Business Problem
Retail businesses generate massive amounts of transactional data daily, making it difficult to monitor performance across products, outlet locations, and customer satisfaction levels.
Key questions include:
Which product categories generate the highest sales?
Which outlet types perform best?
How do outlet sizes impact revenue?
Which locations contribute the most to total sales?
What is the overall customer satisfaction level?
🎯 Goal of the Dashboard
To provide an interactive analytics solution that:
Tracks overall sales performance.
Identifies top-selling product categories.
Evaluates outlet performance by size and location.
Monitors customer ratings and inventory distribution.
Supports strategic business decisions with real-time insights.
